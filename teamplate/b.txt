Objetivo

Implementar o redesign do painel atual sem reescrever o sistema do zero, mantendo as funcionalidades existentes e reorganizando a experiência por camadas de uso, consistência visual e conforto operacional.

A estratégia deve ser de refatoração incremental, preservando a lógica atual do produto, mas substituindo gradualmente layout, componentes, padrões visuais e fluxos de interação.

1. Estratégia de implementação
Direção geral

A implementação não deve começar por páginas isoladas.
Ela deve começar pela base visual e estrutural do sistema.

A ordem correta é:

Design tokens e tema
Layout global
Sidebar e page headers
Componentes reutilizáveis
Dashboard
Client hub
Board
Posting calendar
Ajustes transversais
Refinamento e performance
Estratégia de refatoração

A abordagem ideal é “estrangular o antigo aos poucos”, sem quebrar o sistema existente.

Regra
manter hooks, stores, loaders e lógica de dados
substituir primeiro a camada de apresentação
migrar por feature, não por arquivo aleatório
criar componentes novos e plugar onde já existe valor
evitar refactor estrutural do domínio logo no início

6. Dark/light mode
Implementação

Usar next-themes ou estratégia equivalente.

Provider
"use client";

import { ThemeProvider as NextThemesProvider } from "next-themes";

export function ThemeProvider({ children }: { children: React.ReactNode }) {
  return (
    <NextThemesProvider attribute="class" defaultTheme="dark" enableSystem>
      {children}
    </NextThemesProvider>
  );
}
Toggle
"use client";

import { Moon, Sun } from "lucide-react";
import { useTheme } from "next-themes";

export function ThemeToggle() {
  const { theme, setTheme } = useTheme();

  return (
    <button
      className="inline-flex h-10 w-10 items-center justify-center rounded-md border border-border bg-surface text-textPrimary transition hover:bg-surfaceSecondary"
      onClick={() => setTheme(theme === "dark" ? "light" : "dark")}
    >
      {theme === "dark" ? <Sun size={18} /> : <Moon size={18} />}
    </button>
  );
}

7.3 StatusChip
import { cn } from "@/lib/utils";

type StatusVariant =
  | "default"
  | "success"
  | "warning"
  | "danger"
  | "info"
  | "muted";

export function StatusChip({
  label,
  variant = "default",
}: {
  label: string;
  variant?: StatusVariant;
}) {
  const styles = {
    default: "bg-surfaceSecondary text-textPrimary",
    success: "bg-successSoft text-success",
    warning: "bg-warningSoft text-warning",
    danger: "bg-dangerSoft text-danger",
    info: "bg-infoSoft text-info",
    muted: "bg-surfaceSecondary text-textSecondary",
  };

  return (
    <span
      className={cn(
        "inline-flex items-center rounded-full px-2.5 py-1 text-xs font-medium",
        styles[variant]
      )}
    >
      {label}
    </span>
  );
}
